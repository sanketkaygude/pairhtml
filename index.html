📋 Upstox Access Token — Step-by-Step Guide
// Daily token generation for live NSE market data
1

Create Upstox Developer App

Go to developer.upstox.com → Login with your Upstox trading account → Click Create App. Set Redirect URL as https://127.0.0.1/ (or any URL you control). Note down your API Key and API Secret.

2

Open Auth URL in Browser

Paste this URL in your browser (replace YOUR_API_KEY):
https://api.upstox.com/v2/login/authorization/dialog?response_type=code&client_id=YOUR_API_KEY&redirect_uri=https://127.0.0.1/

Login with your Upstox credentials and click Allow. You'll be redirected to a URL like:
https://127.0.0.1/?code=AUTH_CODE_HERE
Copy the code value from that URL.

3

Exchange Code for Access Token

Run this in terminal / Postman / any REST client:

POST https://api.upstox.com/v2/login/authorization/token
Content-Type: application/x-www-form-urlencoded

Body params:
code=<AUTH_CODE>
client_id=<API_KEY>
client_secret=<API_SECRET>
redirect_uri=https://127.0.0.1/
grant_type=authorization_code

Response will contain "access_token": "ey..." — copy this value.

4

Paste Token in Dashboard

Paste the access_token value in the Access Token field above and click ⚡ Connect. Token is valid until midnight IST. You must regenerate daily before market open.

5

Quick Daily Method (Recommended)

Save a Python script on your phone/PC:
Install: pip install upstox-python-sdk
Or use Upstox Docs → Interactive Token Generator to get token in 2 clicks.

⚠ Token expires at midnight IST every day — regenerate each morning before 9:15 AM.
⚠ Never share your access token publicly or commit it to GitHub.
⚠ CORS: Direct browser calls to Upstox may fail on some networks. If so, use a local proxy or the Upstox SDK.
